Как работает программа:

    Создает или подключается к существующему блоку разделяемой памяти

    Использует именованный семафор для синхронизации

    Предоставляет интерактивное меню для работы со стеком

    Можно запускать несколько экземпляров программы - они будут работать с одним стеком
