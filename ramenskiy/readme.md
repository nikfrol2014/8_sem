# P1 Библиотека PTHREAD. Разделяемая память. Семафоры.

1. Написать программу, вычисляющую сумму элементов n
последовательностей вещественных чисел, находящихся в n файлах,
имена которых задаются в командной строке. Ответ должен быть
записан в файл res.txt. Программа запускает n процессов, вычисляющих
ответ для каждого из файлов и прибавляющих его к результату,
находящемуся в выходном файле. Взаимное исключение при доступе к
файлу обеспечивается с помощью семафора.

2. Написать реализацию стека строк в разделяемой памяти. При запуске
программа создает блок разделяемой памяти (если его еще нет) или
присоединяется к существующему блоку. Программа должна
обеспечивать основные функции работы со стеком (добавить и удалить
элемент) и возможность запуска себя во многих экземплярах.

3. Написать multithread-функцию, получающую в качестве аргументов n ×
n массив a вещественных чисел, целое число n, номер задачи (thread) k,
общее количество задач (threads) p, и возвращающую ненулевое
значение, если массив a симметричен (т. е. aij = aji), 0 в противном
случае. При этом должна быть обеспечена равномерная загрузка всех
задач. Основная программа должна вводить числа p, n и массив a (из
файла или по заданной формуле), запускать задачи, вызывать эту
функцию и выводить на экран результат ее работы.
Имя файла с матрицей передается через командную строку. Формат
файла с матрицей: Размерность_матрицы Количество_потоков Данные

n массив a вещественных чисел, целое число n, 
общее количество задач (threads) p, и заменяющую матрицу a на матрицу (a+at)/2,

файла или по заданной формуле), запускать задачи, вызывать эту
